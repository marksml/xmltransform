Xmltransform

If the MS Excel Import for multiple similar XML files is not working for you then this tool might be helpful. XmlTransform is a small tool which allows you to select a folder with xml documents and get them all serialized and flattened to a result file in comma-separated format that can be easily imported in MS Excel, Libre Office or Open Office.

Project Overview

A small Swing-based Java tool which allows you to select a folder with xml documents and get them all serialized and flattened to a result file in comma-separated format. The XML tag names in the XML documents are retrieved and added as first row in the csv file and can therefore be reused as column names in Excel. The project was created to allow easy import of many XML docs of the same type in one worksheet in Excel.
